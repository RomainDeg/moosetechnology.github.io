## Moose is a community. <span class="text-blue-500">Get involved.</span>

The Moose project started in 1996 at the University of Bern.  
Its original goal was to build a language-independent meta-model for software analysis.  
Since then, it grew to become an open platform for various types of software and data analyses.  
Its development is currently supported by several research groups, startups, and contributors.

- <span class="text-blue-500">Join</span> the [mailing list](https://sympa.inria.fr/sympa/subscribe/moose).
- <span class="text-blue-500">Follow</span> [@moosetechnology](https://twitter.com/moosetechnology).
- <span class="text-blue-500">Check</span> the code from [GitHub](https://github.com/moosetechnology/Moose).
- <span class="text-blue-500">Raise</span> an [issue](https://github.com/moosetechnology/Moose/issues).
- <span class="text-blue-500">Write</span> your story in the [blog](/blog/).
- <span class="text-blue-500">Code & Learn</span> during the [Moose Dojo](https://github.com/moosetechnology/Famix/projects/1#column-11845458).
